metadata.json

To submit analysis results to JAMO (i.e. archive and index the results), you must first create a template as described here then create a file called 'metadata.json' which contains the actual data from the analysis you wish to save.

Format

{
    "outputs": [ { 
                    "file": //the path relative to the root of the analysis directory
                    "label": //the type this file is that matches in the template
                    "metadata": {"key":value,... }//a dictonary that contains the metadata for this file
                 },...
               ],
    "metadata": { "key": value,...},//a dictonary that contains the metadata for this analysis
    "inputs": [ // the path to all the inputs to this analysis
                 "/global/dna/dm_archive/sdm/1234.fastq.gz",...
              ],
}
