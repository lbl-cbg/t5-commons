Adding more metadata to a file

It is possible for users, with the correct permissions, to add more metadata to any record. For example if some post processing process occurs on a file that generates some stats for said file, then it would be nice to add those stats to the metadata for that file.

Make the call

To make the call you need to know the metadata id of the file you wish to update, and the metadata you wish to add. 
from sdm_curl import Curl
curl = Curl('https://sdm-dev.jgi-psf.org:8034',appToken='TOKEN')

curl.put('api/metadata/update/%s'%metadata_id,metadata={
        'number_of_reads':100
    })

This call will put the metadata you passed in into a subdocument identified by your username
