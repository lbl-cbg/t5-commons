JAMO

JAMO is JGI Archive and Metadata Organizer. It is a:
  • Managed repository of files/data
  • Hierarchical Metadata system
  • Data provenance system
  • Archive and retrieval system
  • File lifecycle system
  • External system metadata broker
  • User definable custom search and result system
JAMO manages /global/dna/dm_archive (a.k.a. DnA or the Data 'n Archive file system at JGI). JAMO automatically stores files on NERSC's HPSS tape system if requested and will manage the search and retrieval of data on disk and tape.
